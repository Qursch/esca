# esca
A project for WaffleHacks '21.


## Installation

Clone this repository.

```bash
git clone https://github.com/Qursch/esca.git
```

Use [npm](https://www.npmjs.com/) to install the necessary packages.

```bash
cd esca
npm install
```

Next add and set the following `.env` variables.

```bash
MONGO_URI=...
SESSION_SECRET=...
```
